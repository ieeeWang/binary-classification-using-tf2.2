# binary-classification-using-tf2.2
Implement a binary classification using keras, medium-level APIs (i.e., subclassing tf.keras.Model) and low-level APIs (subclassing tf.Module, customized loss and metric functions). 

Note: issubclass(tf.keras.Model,  tf.Module) = True.
